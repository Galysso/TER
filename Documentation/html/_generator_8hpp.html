Detailed Description

Generate a connected graph G=(V,E)
Exemple of data format
Vi is the vertex indexes, VCij is the indexes of vertices connected to Vi :
|V|
V0 |V0| VC00 VC01 VC02 VC03
V1 |V1| VC10 VC11 VC12 VC13 VC14 VC15
V2 |V2| VC20
V3 |V3| VC30 VC31
V4 |V4| VC40 VC41 VC42
.

Author
Jocelin C.
Parameters
1(card) : graph cardinality, |V| (card > 1)
2(min) : minimum degree of vertices ((min > 0) AND (min < card))
3(max) : maximum degree of vertices ((max > min-1) AND (min < card))
4(file) : name of the generated file
